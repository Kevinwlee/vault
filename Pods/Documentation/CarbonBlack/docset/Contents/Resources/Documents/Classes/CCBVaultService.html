Class Methods

sharedService

The shared CCBVaultSerivce should be used to persist data to the Carbon Vault API. This class provides methods for creating data in containers. You can think of a container as a buck for common data. Once you persist data, you can retrieve it by id. You can also retrieve all data in a specific container, and you can get a listing of all your containers.

+ (CCBVaultService *)sharedService

Return Value

the static shared instance of the the Vault Service

Discussion

The shared CCBVaultSerivce should be used to persist data to the Carbon Vault API. This class provides methods for creating data in containers. You can think of a container as a buck for common data. Once you persist data, you can retrieve it by id. You can also retrieve all data in a specific container, and you can get a listing of all your containers.

createItem:inContainer:completion:

If you need to store data in the Carbon Vault, you will create the data in a container.

- (void)createItem:(NSDictionary *)item inContainer:(NSString *)containerName completion:(vaultCompletionBlock)completionBlock

Parameters

item

= the NSDictionary representation of the item you want to persist

containerName

is the name of the container that will be used to store similar data

completionBlock

is executed after the create operation has completed on the server.

Discussion

If you need to store data in the Carbon Vault, you will create the data in a container.
